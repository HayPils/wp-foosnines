== Description ==

Builds a site that tracks 5NINES foosball events and player stats.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-foosnines.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
